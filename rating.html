My Rating System


My rating system isn't exactly standard. For this reason, I've included this page which describes how the films are rated, and how the two area ratings are combined to achieve an "Overall" rating.

Each movie is given an "individual" rating. One "Technical," and one "Fun" rating. Technical rating is based on overall technical quality of the film. This includes aspects like narrative, acting, directing, cinematography, etc. Secondly, the "Fun" rating is simply based on how much I enjoyed the film. Each area is given a letter, which then corresponds to a value.

To achieve the "Overall" rating, the values of the two areas are averaged, and then I use this "Overall" table to find which letter corresponds to the value the average produced.
